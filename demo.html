How to Use

  1. Encode your payment data as base64 JSON
  2. Add it as the invoice URL parameter
  3. Users will see payment details with QR code
  4. Users manually send the exact amount using their wallet

Invoice Structure

{ "chainId": number, // Chain ID (137 for Polygon, 1 for Ethereum, etc.) "tokenAddress": string | null, // Token contract address, null for native tokens "amount": number, // Amount in human readable format "address": "0x..." // Recipient address }

Encoding Example (Node.js)

const invoiceData = { chainId: 137, // Polygon tokenAddress: "0x2791Bca1f2de4661ED88A30C99A7a9449Aa84174", // USDC on Polygon amount: 15.634771, address: "0x8daA560362b330b71E893fD1AD1B24A640d8022A" }; const encodedInvoice = Buffer.from(JSON.stringify(invoiceData)).toString("base64"); const widgetUrl = `https://example.com/widget?invoice=${encodedInvoice}`;
